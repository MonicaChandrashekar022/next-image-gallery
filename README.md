# Next.js Responsive Image Gallery

A fully **responsive image gallery** with a **modal slider** built using **Next.js 16**, **TypeScript**, and **Tailwind CSS**. This project demonstrates fetching images from a public API, smooth animations, lazy loading, and a polished UI experience.

## üöÄ Features

- Fetches images from **[Picsum Photos API](https://picsum.photos/v2/list)**  
- Responsive **grid layout** for desktop, tablet, and mobile  
- **Modal slider** for full-size viewing  
- Smooth animations with **Framer Motion**  
- **Lazy loading** for optimized performance  
- Keyboard & mobile swipe navigation  
- Light/Dark theme toggle  
- Error handling and loading placeholders
